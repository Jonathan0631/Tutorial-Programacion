📋 Listas

Las listas permiten guardar varios valores en una sola variable.

🔹 Básico

Ejemplo sencillo de una lista:

lista = [10, 20]
print(lista)

Este código muestra todos los valores guardados en la lista.

🔹 Intermedio

Ahora agregamos un nuevo valor a la lista:

lista = [10, 20]
lista.append(30)
print(lista)

El método append sirve para agregar un elemento al final de la lista.

🔹 Ejercicio

Intenta agregar un número a una lista.

🔹 Solución

lista = [5, 10]
lista.append(15)
print(lista)

Así se agrega un nuevo número a la lista correctamente.
