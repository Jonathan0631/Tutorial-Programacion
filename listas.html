📋 Listas

Una lista es una estructura que permite almacenar varios elementos en una sola variable, facilitando la organización de datos.

Ejemplo

Código

frutas = ["manzana", "banana", "uva"]
print(frutas)

Explicación

Se crea una lista llamada frutas con varios elementos y luego se muestra en pantalla.

Salida

['manzana', 'banana', 'uva']

🔹 Básico

Ejemplo sencillo de una lista:

Código

lista = [10, 20]
print(lista)

Explicación

Se crea una lista con valores numéricos y luego se muestra completa en pantalla.

Salida

[10, 20]

🔹 Intermedio

Ahora añadimos un elemento:

Código

lista = [10, 20]
lista.append(30)
print(lista)

Explicación

El método append agrega un nuevo elemento al final de la lista y luego se muestra el resultado.

Salida

[10, 20, 30]

🔹 Ejercicio

Intenta agregar un número a una lista.

🔹 Solución

Código

lista = [5, 10]
lista.append(15)
print(lista)

Explicación

Se crea una lista inicial y se agrega un nuevo valor utilizando append, mostrando el resultado final.

Salida

[5, 10, 15]

En resumen, las listas permiten almacenar y organizar múltiples datos de manera sencilla.
