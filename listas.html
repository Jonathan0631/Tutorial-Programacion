📋 Listas

Una lista es como una mochila donde puedes guardar varios datos.

🔹 Básico

Ejemplo sencillo de una lista:

lista = [10, 20]
print(lista)

Explicación del código:

  • Se crea una lista
  • Se muestran los datos

Este código muestra los valores.

🔹 Intermedio

Ahora añadimos un elemento:

lista = [10, 20]
lista.append(30)
print(lista)
  • append: agrega un nuevo valor

El método append sirve para agregar un elemento al final de la lista.

🔹 Ejercicio

Intenta agregar un número a una lista.

🔹 Solución

Así se agrega un valor correctamente.

lista = [5, 10]
lista.append(15)
print(lista)

Explicación del código:

  • Se crea una lista con números
  • append: agrega un nuevo valor
  • print: muestra la lista

Este código agrega un nuevo número y lo muestra en pantalla.

En resumen, las listas ayudan a organizar datos fácilmente.
