📂 Archivos

Los archivos sirven para guardar información en el computador, como escribir en un cuaderno para usarlo después.

Ejemplo

Código

archivo = open("datos.txt", "w")
archivo.write("Hola mundo")
archivo.close()

Explicación

Este código crea un archivo en modo escritura llamado datos.txt y guarda el texto "Hola mundo".

Salida

Se crea el archivo datos.txt con el contenido: Hola mundo

🔹 Básico

Ejemplo sencillo para guardar información en un archivo:

archivo = open("texto.txt", "w")
archivo.write("Hola")
archivo.close()

Explicación del código:

  • open("texto.txt", "w"): crea un archivo para escribir
  • write("Hola"): guarda la palabra Hola
  • close(): cierra el archivo

🔹 Intermedio

Ahora vamos a leer lo que guardamos en el archivo:

archivo = open("texto.txt", "r")
contenido = archivo.read()
print(contenido)
archivo.close()

Explicación del código:

  • "r": significa leer el archivo
  • read(): obtiene el contenido
  • print: muestra el contenido en pantalla

🔹 Ejercicio

Intenta guardar un nombre en un archivo.

🔹 Solución

archivo = open("nombre.txt", "w")
archivo.write("Juan")
archivo.close()

Explicación del código:

  • open("nombre.txt", "w"): crea un archivo para escribir
  • write("Juan"): guarda el nombre en el archivo
  • close(): cierra el archivo

Este código guarda el nombre en un archivo correctamente.

En resumen, los archivos permiten guardar información para usarla después cuando la necesites.
