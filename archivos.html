📂 Archivos

Los archivos permiten guardar información en el computador y también leerla cuando sea necesario.

🔹 Básico

Ejemplo sencillo para guardar texto en un archivo:

archivo = open("texto.txt", "w")
archivo.write("Hola")
archivo.close()

Este código crea un archivo llamado "texto.txt" y guarda la palabra "Hola".

🔹 Intermedio

Ahora leemos el contenido de un archivo:

archivo = open("texto.txt", "r")
contenido = archivo.read()
print(contenido)
archivo.close()

Este código abre el archivo y muestra su contenido en pantalla.

🔹 Ejercicio

Intenta guardar un nombre en un archivo.

🔹 Solución

archivo = open("nombre.txt", "w")
archivo.write("Juan")
archivo.close()

Así se guarda correctamente un nombre en un archivo.
