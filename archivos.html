📂 Archivos

Los archivos permiten guardar información en el computador, de forma similar a escribir en un cuaderno para usarla después.

Ejemplo

Código

archivo = open("datos.txt", "w")
archivo.write("Hola mundo")
archivo.close()

Explicación

Este código crea un archivo llamado datos.txt en modo escritura y guarda el texto "Hola mundo".

Salida

Se crea el archivo datos.txt con el contenido: Hola mundo

🔹 Básico

Ejemplo sencillo para guardar información en un archivo:

Código

archivo = open("texto.txt", "w")
archivo.write("Hola")
archivo.close()

Explicación

Este código crea un archivo llamado texto.txt en modo escritura y guarda el texto "Hola".

Salida

Se crea el archivo texto.txt con el contenido: Hola

🔹 Intermedio

Ahora vamos a leer lo que guardamos en el archivo:

Código

archivo = open("texto.txt", "r")
contenido = archivo.read()
print(contenido)
archivo.close()

Explicación

Este código abre el archivo en modo lectura, obtiene su contenido y lo muestra en pantalla.

Salida

Hola

🔹 Ejercicio

Intenta guardar un nombre en un archivo.

🔹 Solución

Código

archivo = open("nombre.txt", "w")
archivo.write("Juan")
archivo.close()

Explicación

Se crea un archivo llamado nombre.txt y se guarda el texto "Juan" en su interior.

Salida

Se crea el archivo nombre.txt con el contenido: Juan

En resumen, los archivos permiten guardar información para usarla posteriormente cuando sea necesario.
