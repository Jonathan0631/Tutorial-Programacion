⚙️ Funciones

Las funciones permiten reutilizar código, es decir, ejecutar varias veces el mismo conjunto de instrucciones sin repetirlo.

🔹 Básico

Ejemplo sencillo de una función:

def saludar():
    print("Hola")
saludar()

Esta función muestra la palabra "Hola" cuando se ejecuta.

🔹 Intermedio

Ahora usamos una función con un dato (nombre):

def saludar(nombre):
    print("Hola", nombre)
saludar("Juan")

Esta función muestra "Hola Juan" porque usamos un nombre como parámetro.

🔹 Ejercicio

Intenta crear una función que muestre el mensaje "Bienvenido".

🔹 Solución

def mensaje():
    print("Bienvenido")
mensaje()

Así se crea una función que muestra el mensaje correctamente.
