⚙️ Funciones

Las funciones son bloques de código que permiten ejecutar una acción cuando se necesite, evitando repetir instrucciones varias veces.

Ejemplo

Código

def saludar(nombre):
    return "Hola " + nombre

print(saludar("Juan"))

Explicación

Se define una función llamada saludar que recibe un nombre y devuelve un saludo. Luego se llama a la función y se muestra el resultado.

Salida

Hola Juan

🔹 Básico

Ejemplo sencillo de una función:

Código

def saludar():
    print("Hola")

saludar()

Explicación

Se define una función sin parámetros que muestra el mensaje "Hola" cuando es llamada.

Salida

Hola

🔹 Intermedio

Ahora usamos una función con parámetro:

Código

def saludar(nombre):
    print("Hola", nombre)

saludar("Juan")

Explicación

La función recibe un nombre como parámetro y lo utiliza para mostrar un saludo personalizado.

Salida

Hola Juan

🔹 Ejercicio

Intenta crear una función que diga "Bienvenido".

🔹 Solución

Código

def mensaje():
    print("Bienvenido")

mensaje()

Explicación

Se crea una función llamada mensaje que muestra un texto en pantalla cuando es ejecutada.

Salida

Bienvenido

En resumen, las funciones permiten organizar y reutilizar código de manera eficiente.
