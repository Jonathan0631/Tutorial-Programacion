⚙️ Funciones

Las funciones son como un botón que guarda una acción para usarla cuando quieras.

Esto ayuda a no repetir el código muchas veces.

Ejemplo

Código

def saludar(nombre):
    return "Hola " + nombre

print(saludar("Juan"))

Explicación

Se define una función llamada saludar que recibe un nombre y devuelve un saludo. Luego se llama a la función y se muestra el resultado.

Salida

Hola Juan

🔹 Básico

Ejemplo sencillo de una función:

def saludar():
    print("Hola")
saludar()

Explicación del código:

  • def: crea una función
  • saludar(): nombre de la función
  • print: muestra texto

Este código muestra un saludo.

🔹 Intermedio

Ahora usamos un nombre:

def saludar(nombre):
    print("Hola", nombre)
saludar("Juan")

Explicación del código:

  • La función recibe un nombre
  • Luego muestra el mensaje con ese nombre

Este código mostrará: Hola Juan

🔹 Ejercicio

Intenta crear una función que diga "Bienvenido".

🔹 Solución

def mensaje():
    print("Bienvenido")
mensaje()

Así se crea una función correctamente.

En resumen, las funciones ayudan a reutilizar código fácilmente.
