⚙️ Funciones

Las funciones permiten reutilizar código.

🔹 Básico

def saludar():
    print("Hola")

saludar()

🔹 Intermedio

def saludar(nombre):
    print("Hola", nombre)

saludar("Juan")

🔹 Ejercicio

Crear una función que muestre "Bienvenido"

🔹 Solución

def mensaje():
    print("Bienvenido")

mensaje()
