🔁 Ciclos

Un ciclo es una forma de repetir una acción varias veces sin escribir lo mismo muchas veces.

Ejemplo

Código

for i in range(5):
    print(i)

Explicación

Este código usa un ciclo for para recorrer números del 0 al 4 y mostrarlos en pantalla.

Salida

0
1
2
3
4

🔹 Básico

Ejemplo sencillo:

for i in range(3):
    print("Hola")

Explicación del código:

  • for: indica que se repetirá algo
  • range(3): significa repetir 3 veces
  • print: muestra el mensaje

Este código repite la palabra "Hola".

🔹 Intermedio

Ahora veremos cómo mostrar números uno por uno:

for i in range(1, 6):
    print(i)

Explicación del código:

  • El ciclo recorre los números
  • Muestra cada número en pantalla

Este código muestra los números del 1 al 5.

🔹 Ejercicio

Intenta hacer un ciclo que muestre números del 1 al 3.

🔹 Solución

Esta es la forma correcta de resolver el ejercicio.

for i in range(1, 4):
    print(i)

Este código muestra los números del 1 al 3 correctamente.

En resumen, los ciclos sirven para repetir tareas de forma fácil y rápida.
