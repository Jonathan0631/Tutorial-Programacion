🔁 Ciclos

Un ciclo es una estructura que permite repetir una acción varias veces sin necesidad de escribir el mismo código repetidamente.

Ejemplo

Código

for i in range(5):
    print(i)

Explicación

Este código utiliza un ciclo for para recorrer números del 0 al 4 y mostrarlos en pantalla.

Salida

0
1
2
3
4

🔹 Básico

Ejemplo sencillo:

Código

for i in range(3):
    print("Hola")

Explicación

El ciclo se ejecuta tres veces y en cada repetición muestra el mensaje "Hola" en pantalla.

Salida

Hola
Hola
Hola

🔹 Intermedio

Ahora veremos cómo mostrar números uno por uno:

Código

for i in range(1, 6):
    print(i)

Explicación

El ciclo recorre los números del 1 al 5 y muestra cada uno en pantalla.

Salida

1
2
3
4
5

🔹 Ejercicio

Intenta hacer un ciclo que muestre números del 1 al 3.

🔹 Solución

Código

for i in range(1, 4):
    print(i)

Explicación

Este ciclo genera números del 1 al 3 y los muestra en pantalla.

Salida

1
2
3

En resumen, los ciclos permiten repetir tareas de forma automática y eficiente.
