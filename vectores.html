📊 Vectores

Un vector es como una caja donde puedes guardar varios datos juntos.

Por ejemplo, puedes guardar números o nombres.

🔹 Básico

Ejemplo sencillo de un vector:

numeros = [1, 2, 3]
print(numeros)
  • [ ]: representa una lista
  • print: muestra los datos

Este código muestra los elementos.

🔹 Intermedio

Ahora recorremos la lista:

numeros = [1, 2, 3]
for n in numeros:
    print(n)

Explicación del código:

  • El ciclo recorre la lista
  • Muestra cada elemento uno por uno

Este ciclo muestra cada número del vector uno por uno.

🔹 Ejercicio

Intenta mostrar colores en una lista.

🔹 Solución

Así se recorren los datos.

colores = ["rojo", "azul", "verde"]
for c in colores:
    print(c)

Explicación del código:

  • Se crea una lista de colores
  • El ciclo recorre cada color
  • Muestra cada elemento en pantalla

Este código muestra todos los colores uno por uno.

En resumen, los vectores permiten trabajar con varios datos.
