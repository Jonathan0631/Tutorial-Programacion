📊 Vectores

Un vector es una estructura que permite almacenar varios datos en una sola variable, similar a una lista.

Ejemplo

Código

numeros = [10, 20, 30]
print(numeros[1])

Explicación

Se crea un vector con varios números y se accede a la posición 1 para mostrar ese valor.

Salida

20

🔹 Básico

Ejemplo sencillo de un vector:

Código

numeros = [1, 2, 3]
print(numeros)

Explicación

Se crea un vector con varios elementos y luego se muestra completo en pantalla.

Salida

[1, 2, 3]

🔹 Intermedio

Ahora recorremos el vector:

Código

numeros = [1, 2, 3]
for n in numeros:
    print(n)

Explicación

El ciclo recorre cada elemento del vector y muestra los valores uno por uno en pantalla.

Salida

1
2
3

🔹 Ejercicio

Intenta mostrar los elementos de una lista de colores.

🔹 Solución

Código

colores = ["rojo", "azul", "verde"]
for c in colores:
    print(c)

Explicación

Se crea una lista de colores y se recorre utilizando un ciclo para mostrar cada elemento en pantalla.

Salida

rojo
azul
verde

En resumen, los vectores permiten almacenar y trabajar con múltiples datos de forma organizada.
