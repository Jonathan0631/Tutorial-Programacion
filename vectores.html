📊 Vectores

Un vector es una lista de datos que permite guardar varios valores en una sola variable.

🔹 Básico

Ejemplo sencillo de un vector:

numeros = [1, 2, 3]
print(numeros)

Este código muestra todos los números del vector.

🔹 Intermedio

Ahora recorremos el vector para mostrar cada elemento:

numeros = [1, 2, 3]
for n in numeros:
    print(n)

Este ciclo muestra cada número del vector uno por uno.

🔹 Ejercicio

Intenta mostrar los elementos de una lista de colores.

🔹 Solución

colores = ["rojo", "azul", "verde"]
for c in colores:
    print(c)

Así se muestran todos los elementos de la lista correctamente.
