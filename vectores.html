📊 Vectores

Un vector es una lista de datos.

🔹 Básico

numeros = [1, 2, 3]
print(numeros)

🔹 Intermedio

numeros = [1, 2, 3]

for n in numeros:
    print(n)

🔹 Ejercicio

Mostrar los elementos de una lista de colores

🔹 Solución

colores = ["rojo", "azul", "verde"]

for c in colores:
    print(c)
